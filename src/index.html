Uppgift

Inför moment 1 ska vi skapa en grundstruktur för webbutveckling kompletterat med automatisering av vissa uppgifter. Det ska vara för en webbplats som innehåller HTML, CSS, JavaScript och bilder. Jag kommer använda Node.js tillsammans med Parcel för automatisera dessa uppgifter. En package.json-fil ska finnas med alla installerade paket där det ska gå att återställa applikationen med 'npm install'. De uppgifter som skall automatiseras är minifiering av CSS och JavaScript, minifiering av HTML, optimering av bilder, bakåtkompabilitet i genererade filer och kopiering av arbetsfiler till mapp för publicering med optimerade filer. Sedan skapas en demowebbplats som ska innehålla en huvudmeny med minst tre undersidor, en för förklaring av uppgiften, en som visar minst 5 bilder, och en som beskriver arbetsprocessen.

Arbetsprocessen

Jag började med att skapa en mapp för moment 1. Genom kommandot npm init, så initierade jag ett nytt npm projekt där jag fyllde i mitt namn och beskrivning av projektet. Detta skapade ett package.json fil i mappen för moment 1. Jag initierade även ett github repo till mappen och skapade en ny branch för utveckling (development) tanken är att göra allt arbete samt tester innan jag mergar till main branch för publicering. Jag skapade även en gitignore fil där jag lägger till undantag som jag inte vill spara i mitt git repo.

Tillbaka till npm projektet, så installerade jag parcell med kommandot npm install parcell --save -d. vilket skapade en ny mapp vid namn "node_modules" som jag lade till i gitignore. vilket även uppdaterade package.json filen med så kallade dependencies vilket betyder att när kommandot npm install körs, så installeras samtliga paket (i detta fall parcell) jag skapade en mapp vid namn src där jag lägger min index.html fil, jag skapar mappar för JavaScript, CSS och Bilder och det är i denna mapp jag gör alla mina ändringar. Jag la även till scripts i package.jason filen för start och build så jag kan använda kommandon som npm run start/build.

Jag skapade en enkel html sida med 3 sektioner, samt en fungerande hamburgarmeny som fälls ut efter klick med hjälp av JavaScript. jag fyllde i texten som krävdes inför momentet länkade till samtliga bilder och la till stil med hjälp av CSS.

Bilder konverteras från png till webp och en bildbredd på 1200 pixlar med hjälp av parcels inbygga bildverktyg När allt var klart kunde jag köra kommandot 'npm run build' och parcell skötte minifiering och transpilering av samtliga filer och flyttade dessa optimerade filer till dist-mappen
